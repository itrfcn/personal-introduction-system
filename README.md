# 个人介绍系统

一个功能完整的个人介绍系统，包含前台展示和后台管理功能，支持文章发布、个人信息管理、旗下站点管理和社交图标管理等功能。

## 功能特点

### 前台功能
- **首页**：展示最新文章和个人信息摘要
- **个人介绍**：详细展示个人信息、教育背景、工作经验和技能专长
- **文章列表**：展示所有已发布的文章
- **文章详情**：查看单篇文章的完整内容
- **旗下站点**：展示所有旗下站点信息

### 后台功能
- **用户认证**：安全的后台登录验证
- **仪表盘**：系统概览和快速操作
- **文章管理**：文章的增删改查操作，支持发布/草稿状态
- **个人信息管理**：编辑个人简介、技能、教育和工作经验
- **旗下站点管理**：站点的增删改查操作
- **社交图标管理**：社交账号图标的增删改查操作
- **系统设置**：网站基本信息和系统配置管理
- **修改密码**：安全的密码修改功能

## 技术栈

- **前端技术**：HTML5、CSS3、JavaScript、Font Awesome
- **后端技术**：PHP 7.0+
- **数据存储**：支持两种模式
  - JSON文件存储（适合小型网站）
  - MySQL 5.7+数据库存储（适合大型网站）

## 安装步骤

### 1. 启动服务器

使用PHP内置服务器或其他Web服务器（如Apache、Nginx）启动项目：

```bash
# 使用PHP内置服务器
php -S localhost:8000
```

### 2. 自动安装向导

- 访问 `http://localhost:8000`，系统将自动跳转到安装页面
- 按照安装向导完成安装：

#### 2.1 选择存储模式
- **文件存储 (data文件夹)**：使用JSON文件存储数据，简单易用，适合小型网站
- **数据库存储 (MySQL)**：使用MySQL数据库存储数据，性能更好，适合大型网站

#### 2.2 数据库配置（仅MySQL模式需要）
- 数据库主机：默认 `localhost`
- 数据库端口：默认 `3306`
- 数据库名称：默认 `personal_website`
- 数据库用户名：默认 `root`
- 数据库密码：根据实际情况填写

#### 2.3 管理员信息
- 管理员用户名：设置后台登录用户名
- 管理员密码：设置后台登录密码（至少6个字符）

#### 2.4 网站基本信息
- 网站标题：网站的主要标题
- 网站副标题：网站的次要标题
- 网站URL：网站的访问地址
- 网站描述：网站的简短描述
- 版权信息：网站底部的版权声明
- Logo文字：网站Logo的文字
- Logo图标：使用Font Awesome图标
- 个人简介：网站所有者的简要介绍

### 3. 完成安装

点击「完成安装」按钮，系统将自动创建配置文件并初始化数据。

## 数据存储模式

### JSON文件存储
- 数据存储在 `data/` 目录下的JSON文件中
- 无需额外安装数据库
- 适合小型网站和快速部署

### MySQL数据库存储
- 数据存储在MySQL数据库中
- 性能更好，适合大型网站
- 需要先创建数据库
- 安装时会自动创建表结构

## 重置系统

如果需要重新安装系统，可以执行以下操作：

1. 删除 `data/` 目录下的所有文件（如果使用JSON存储）
2. 或者清空数据库表（如果使用MySQL存储）
3. 删除 `data/install.lock` 文件（如果存在）
4. 再次访问系统，将重新进入安装流程

1. 确保PHP版本不低于7.0
2. 如果使用MySQL存储模式，确保MySQL版本不低于5.7
3. 定期备份数据（JSON文件或数据库）
4. 不要在生产环境使用默认的管理员密码
5. 确保 `data/` 目录有写权限（如果使用JSON存储）
